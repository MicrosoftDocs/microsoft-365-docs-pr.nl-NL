# <a name="evaluate-microsoft-cloud-app-security"></a>Evaluatie van Microsoft Cloud App Security

**Van toepassing op:**
- Microsoft 365 Defender


In dit artikel wordt het proces beschreven voor het inschakelen en Microsoft Cloud App Security en Microsoft 365 Defender. Voordat u dit proces start, moet u [](eval-overview.md) het algemene proces voor het evalueren van Microsoft 365 Defender hebben bekeken en de Microsoft 365 Defender [evaluatieomgeving hebben gemaakt.](eval-create-eval-environment.md) 
<br>

Gebruik de volgende stappen om de beveiliging van microsoft-cloud-apps in te stellen en te piloten.

![Stappen voor het toevoegen van Microsoft Defender Office aan de evaluatieomgeving van Defender](../../media/defender/m365-defender-office-eval-steps.png)



|Stap  |Omschrijving  |
|---------|---------|
|[Architectuurvereisten en belangrijke concepten bekijken](eval-defender-mcas-architecture.md)    | Begrijp de Cloud App Security architectuur en hoe deze integreert met Microsoft 365 Defender, Microsoft Defender voor Eindpunt en Azure Active Directory.        |
|[De evaluatieomgeving inschakelen](eval-defender-mcas-enable-eval.md)     | Verbinding maken de portal, configureert u integratie met Defender voor identiteit en/of netwerkapparaten van uw organisatie en begint u cloud-apps te bekijken en te beheren.         |
|[De pilot instellen ](eval-defender-mcas-pilot.md)    | Bereik uw implementatie naar bepaalde gebruikersgroepen, configureer Voorwaardelijke toegangsbeheer voor apps en probeer zelfstudies voor het beveiligen van uw omgeving.       |
