# <a name="data-monitoring-and-reporting-in-the-microsoft-365-security-center"></a>Gegevensbewaking en -rapportage in het Microsoft 365-beveiligingscentrum

De categorie **Gegevens** helpt gebruikersactiviteiten bij te houden die kunnen leiden tot ongeautoriseerde gegevensopenbaarmaking. Dit zijn de herbewerking van bestaande Office 365 DLP-beleidsrapporten plus een DLP-beleidsovereenkomstrapport van derden.

U zien:

* Gebruikers die de meeste bestanden delen vanuit cloud-apps
* Hoeveel DLP-beleidsovereenkomsten hebben plaatsgevonden
* Hoeveel DLP-beleidsregels worden overschreven of false positives worden gerapporteerd
* Hoeveel DLP-beleidsovereenkomsten zijn er gebeurd in cloudservices van derden via Microsoft Cloud App Security
