# <a name="identity-monitoring-and-reporting-in-the-microsoft-365-security-center"></a>Identiteitscontrole en rapportage in het Microsoft 365-Beveiligingscentrum

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


De identiteiten in uw organisatie bewaken en verdacht of riskant gedrag bijhouden. In de categorie **Identities** van **rapporten**kunt u het volgende bijhouden:

* Gebruikers met de meest ontdekte afwijkingen
* Hoeveel gebruikers worden gerapporteerd aan risico per beleid voor voorwaardelijke toegang
* Het aantal globale beheerders in uw organisatie

![Categorie Identities van rapporten pagina](../../media/identities.png)

Voor gebruikers met specifieke detectie kunt u de specifieke waarschuwing verkennen en onderzoek doen in Microsoft Defender-Beveiligingscentrum. Detectie omvat ook afwijkingen, zoals gebruikers die zich aanmelden via onbekende locaties.

Zie [gebeurtenissen van Azure Active Directory Risk](https://docs.microsoft.com/azure/active-directory/reports-monitoring/concept-risk-events)voor een volledige set met risico gebeurtenissen.
