# <a name="microsoft-365-defender-data-security-and-privacy"></a>Microsoft 365 Defender Data Security en privacy

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**Van toepassing op:**
- Microsoft 365 Defender

Microsoft 365 Defender werkt in Microsoft Azure-datacenters in de Europese Unie, het Verenigd Koninkrijk en de Verenigde Staten. Klantgegevens die worden verzameld door de service, worden ter plaatse opgeslagen in (a) de geografische locatie van de Tenant zoals aangegeven tijdens het inrichten of, (b) als Microsoft 365 Defender gebruikmaakt van een ander Microsoft Online-service voor het verwerken van dergelijke gegevens, de geolocatie zoals gedefinieerd door de opslagregels van die andere online service.

Klantgegevens in een begeleid formulier kunnen ook worden opgeslagen in de centrale opslag-en verwerkingssystemen in de Verenigde Staten.


Zie voor meer informatie over de opslag van gegevens en privacy-informatie van de specifieke producten:
- [Microsoft Defender voor gegevensopslag en privacy voor eindpunten](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/data-storage-privacy)
- [Beveiligingsgegevens beveiliging en privacy voor Microsoft Cloud-app](https://docs.microsoft.com/cloud-app-security/cas-compliance-trust)
- [Privacy, beveiliging en doorzichtigheid in Microsoft 365](https://docs.microsoft.com/office365/servicedescriptions/office-365-platform-service-description/privacy-security-and-transparency#advanced-threat-protection)
