# <a name="microsoft-threat-protection-data-security-and-privacy"></a>Gegevensbeveiliging en privacy voor Microsoft Threat Protection

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**Van toepassing op:**
- Microsoft Threat Protection

Microsoft Threat Protection werkt in Microsoft Azure datacenters in de Europese Unie, het Verenigd Koninkrijk en de Verenigde Staten. Klantgegevens die worden verzameld door de service, worden ter plaatse opgeslagen in de geografische locatie van de Tenant zoals aangegeven tijdens de inrichting of, (b) als Microsoft Threat Protection gebruikmaakt van een andere Microsoft Online service voor het verwerken van dergelijke gegevens, de geolocatie zoals gedefinieerd door de regels voor gegevensopslag van die andere online service.

Klantgegevens in een begeleid formulier kunnen ook worden opgeslagen in de centrale opslag-en verwerkingssystemen in de Verenigde Staten.


Zie voor meer informatie over de opslag van gegevens en privacy-informatie van de specifieke producten:
- [Microsoft Defender ATP data storage en privacy](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/data-storage-privacy)
- [Beveiligingsgegevens beveiliging en privacy voor Microsoft Cloud-app](https://docs.microsoft.com/cloud-app-security/cas-compliance-trust)
- [Privacy, beveiliging en doorzichtigheid in Microsoft 365](https://docs.microsoft.com/office365/servicedescriptions/office-365-platform-service-description/privacy-security-and-transparency#advanced-threat-protection)
