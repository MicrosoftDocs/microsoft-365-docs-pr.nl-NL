# <a name="export-case-data-in-advanced-ediscovery"></a>Casegegevens exporteren in Advanced eDiscovery

Er zijn drie manieren om gegevens uit een revisieset te exporteren:

**Downloaden:** Download (met behulp van een browser) een kleine set oorspronkelijke bestanden. Dit is de snelste manier om een kleine set gegevens te exporteren. Met deze methode blijven de oorspronkelijke bestandsnamen behouden. Zie Documenten downloaden [uit een revisieset](download-documents-from-review-set.md)voor meer informatie.

**Exporteren:** Pas aan welke gegevens worden geëxporteerd. Dit omvat het exporteren van bestandsmetagegevens, oorspronkelijke bestanden, tekstbestanden en redacted documents die zijn opgeslagen in een PDF-bestand. Wanneer de geëxporteerde gegevens zijn voorbereid, kunt u de gegevens downloaden van het **tabblad** Exporteren naar een lokale computer. Zie Documenten exporteren [uit een revisieset](export-documents-from-review-set.md) voor meer informatie.

**Toevoegen aan een andere revisieset:** Kopieer gegevens uit één revisieset naar een andere revisieset. Zie Gegevens uit de [ene revisieset toevoegen aan een andere revisieset voor meer informatie.](add-data-to-review-set-from-another-review-set.md)

> [!NOTE]
> In Microsoft 365 worden gegevens gehasht en worden deze hashes ter verificatie in het uitvoerbestand geleverd. Dit wordt aangevuld met auditlogboeken en rapportagefunctionaliteit, zoals verzamelingsstatistieken, load set-rapporten en exportrapporten (inclusief het export load file).
