# <a name="microsoft-cloud-for-enterprise-architects-illustrations"></a>Microsoft Cloud voor Enterprise Architects

Deze Cloud architecturen bieden u informatie over Microsoft-cloudservices, waaronder Microsoft 365, Azure Active Directory (Azure AD), Microsoft intune, Microsoft Dynamics 365 en hybride on-premises en Cloud oplossingen. 

IT-medewerkers en architecten kunnen deze bronnen gebruiken voor het bepalen van de ideale oplossingen voor hun werkbelastingen en om beslissingen te nemen over de kern infrastructuuronderdelen, zoals netwerken, identiteit en beveiliging.
