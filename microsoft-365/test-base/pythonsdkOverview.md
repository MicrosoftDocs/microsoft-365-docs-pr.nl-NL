# <a name="test-base-sdk-for-python"></a>Test Base SDK voor Python

## <a name="overview"></a>Overzicht
De Test Base SDK kan worden gebruikt voor interactie met de Azure-testbasisbron. (dat wil zeggen uw toepassingspakket beheren, pakket maken, pakket bewerken en verwijderen)

Met de SDK, het testoverzicht en het analyseresultaat dat u kunt krijgen, zijn: scriptExecution, betrouwbaarheid, memoryUtilization, cpuUtilization, memoryRegression, cpuRegression.

Met de Test Base SDK kunt u de testbasis integreren in uw CI/CD-pijplijn.

## <a name="client-library"></a>Clientbibliotheek

Installeer het testbasispakket met pip.

~~~
pip install  Microsoft.Testbase
~~~
