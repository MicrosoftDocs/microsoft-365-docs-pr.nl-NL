# <a name="step-5-choose-your-test-options"></a>Stap 5: Kies uw testopties. 

Het tabblad is voor gebruikers die functionele tests willen uitvoeren om aan te geven wanneer de Windows Update-patch moet worden toegepast in de volgorde van het uitvoeren van hun ```Test Options``` functionele testscripts.

![Afbeelding van testopties. Out-of-box of functionele tests](Media/testoptions.png)

Selecteer _**Controleren**_ om naar het volgende tabblad te gaan en de geselecteerde testopties te bekijken.
