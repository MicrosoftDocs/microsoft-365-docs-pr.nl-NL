# <a name="download-perpetual-software-and-product-license-keys"></a>Permanente software en productlicentie codes downloaden

In dit artikel wordt uitgelegd hoe u productcodes voor software en producten kunt downloaden voor een permanente software die via het CSP-programma (Cloud Solution Provider) is gekocht.

## <a name="before-you-begin"></a>Voordat u begint

U moet een globale beheerder zijn om de stappen in dit artikel te kunnen uitvoeren. Zie[Over beheerdersrollen](../add-users/about-admin-roles.md) voor meer informatie.

## <a name="download-software-and-product-license-keys"></a>Software-en productlicentie codes downloaden

1. Ga in het Microsoft 365-Beheercentrum naar de  >  pagina<a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">uw producten</a> factureren.
2. Selecteer op het tabblad **Products** , in de sectie **Software** , de software die u wilt downloaden.
3. Kies op de pagina Details van abonnement, in de sectie **Downloads & Keys** , de **product versie**, de **taal** en de **CPU & bestandstype** en selecteer vervolgens **downloaden**.
4. Selecteer **code kopiëren naar Klembord** als u de sleutel wilt downloaden.
5. Selecteer in het rechterdeelvenster **kopiëren** en sluit het deelvenster.
6. Plak de sleutel in een bestand op een veilige locatie en voer deze in volgens de instructies tijdens de installatie van de software. De sleutel is nodig om de gedownloade software te activeren.
