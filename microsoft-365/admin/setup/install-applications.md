# <a name="install-office-applications"></a>Office-toepassingen installeren

Nu u Microsoft 365 hebt ingesteld, kunt u afzonderlijke Office-toepassingen installeren op uw Mac, PC of mobiele apparaten.
  
Volg deze koppelingen voor informatie over:
  
- Office-toepassingen installeren:  [Office installeren op uw PC of Mac](https://support.microsoft.com/office/4414eaaf-0478-48be-9c42-23adc4716658)

- Andere apps installeren: [project](https://support.microsoft.com/office/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281), [Visio](https://support.microsoft.com/office/install-visio-f98f21e3-aa02-4827-9167-ddab5b025710)of [Skype voor bedrijven](https://support.microsoft.com/office/install-skype-for-business-8a0d4da8-9d58-44f9-9759-5c8f340cb3fb)

- Mobiele apparaten instellen: [configuratie van Microsoft 365 Mobile-Help](https://support.microsoft.com/office/7dabb6cb-0046-40b6-81fe-767e0b1f014f)

- E-mail instellen in Outlook: [Windows](https://support.microsoft.com/office/6e27792a-9267-4aa4-8bb6-c84ef146101b) of [Mac](https://support.microsoft.com/office/6e27792a-9267-4aa4-8bb6-c84ef146101b#PickTab=Outlook_for_Mac)
 
- [Gebruikers upgraden naar de nieuwste apps](upgrade-users-to-latest-office-client.md) 

::: moniker range="o365-21vianet"

Als u Azure Active Directory Premium (AADP) abonnement 1 of abonnement 2 hebt gekocht, komt u in aanmerking voor Microsoft Identity Manager (MIM). Om MIM te downloaden, gaat u naar het [Downloadcentrum](https://www.microsoft.com/zh-cn/download/details.aspx?id=58498).

::: moniker-end
