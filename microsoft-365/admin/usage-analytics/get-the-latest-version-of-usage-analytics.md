# <a name="get-the-latest-version-of-microsoft-365-usage-analytics"></a>Download de nieuwste versie van Microsoft 365-gebruiksanalyses

De sjabloon-app kan meerdere keren per jaar worden vernieuwd met nieuwe gegevens of nieuwe visualisaties. Uw bestaande instantie blijft werken, maar als u de nieuwste versie wilt krijgen, moet een nieuwe instantie worden gemaakt en moeten eventuele aanpassingen worden toegepast op het nieuwe exemplaar. Zie [Microsoft 365-gebruiksanalyses inschakelen](enable-usage-analytics.md).
