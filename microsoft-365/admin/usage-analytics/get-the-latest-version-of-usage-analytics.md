# <a name="get-the-latest-version-of-microsoft-365-usage-analytics"></a>De nieuwste versie van de Microsoft 365 gebruiksanalyse

De sjabloon-app kan meerdere keren per jaar worden vernieuwd met nieuwe gegevens of nieuwe visualisaties. Uw bestaande exemplaar blijft werken, maar als u de nieuwste versie wilt krijgen, moet er een nieuw exemplaar worden gemaakt en moeten eventuele aanpassingen worden toegepast op het nieuwe exemplaar. Zie [Gebruiksanalyse Microsoft 365 inschakelen.](enable-usage-analytics.md)
