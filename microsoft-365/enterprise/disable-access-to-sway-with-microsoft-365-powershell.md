# <a name="disable-access-to-sway-with-powershell-for-microsoft-365"></a>Toegang tot Sway uitschakelen met PowerShell voor Microsoft 365

*Dit artikel is van toepassing op Microsoft 365 Enterprise en Office 365 Enterprise.*

Met de ManageSway.ps1 PowerShell-script kunt u services weergeven en uitschakelen in uw Microsoft 365-organisatie, waaronder Sway. Met dit script worden de procedures geautomatiseerd die in de volgende onderwerpen worden beschreven:
  
- [Licenties en services weergeven met PowerShell](view-licenses-and-services-with-microsoft-365-powershell.md)
    
- [Toegang tot services uitschakelen met PowerShell](disable-access-to-services-with-microsoft-365-powershell.md)
    
U moet de twee bestanden die zijn gekoppeld aan het script downloaden:
  
- Het ManageSway.ps1-script op [https://go.microsoft.com/fwlink/p/?LinkId=785070](https://go.microsoft.com/fwlink/p/?LinkId=785070)
    
- Het Help-bestand voor het script op [https://go.microsoft.com/fwlink/p/?LinkId=785072](https://go.microsoft.com/fwlink/p/?LinkId=785072)
