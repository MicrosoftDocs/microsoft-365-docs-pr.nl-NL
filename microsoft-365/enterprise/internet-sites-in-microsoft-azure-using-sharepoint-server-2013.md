# <a name="internet-sites-in-microsoft-azure-using-sharepoint-server-2013"></a>Internetsites in Microsoft Azure met SharePoint Server 2013

 Internetsites die gebruikmaken SharePoint Server 2013 profiteren door te worden gehost in Azure Infrastructure Services. Dit artikel bevat informatiebronnen voor het ontwerpen en implementeren van deze oplossing.

## <a name="using-azure-infrastructure-services-for-internet-sites"></a>Azure Infrastructure Services gebruiken voor internetsites

Microsoft Azure biedt een aantrekkelijke optie voor het hosten van internetsites op basis van SharePoint Server 2013. De volgende voordelen zijn:

- Focus op het ontwikkelen van een geweldige site in plaats van het bouwen van infrastructuur.

- Flexibiliteit om uw oplossing te schalen op basis van de vraag door deze uit te schalen en in te schalen.

- Betaal alleen voor de resources die u nodig hebt en gebruik.

- Profiteer van Azure Active Directory voor klantaccounts.

- Voeg functies toe die momenteel niet beschikbaar zijn in Microsoft 365, zoals diepgaande rapportage en analyse.
