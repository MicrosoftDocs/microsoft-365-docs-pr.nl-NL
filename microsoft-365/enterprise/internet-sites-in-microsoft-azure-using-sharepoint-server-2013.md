# <a name="internet-sites-in-microsoft-azure-using-sharepoint-server-2013"></a>Internet sites in Microsoft Azure met behulp van SharePoint Server 2013

 Internet sites die gebruikmaken van SharePoint Server 2013, profiteren van de voordelen van Azure-infrastructuur services. Dit artikel bevat informatie over het ontwerpen en implementeren van deze oplossing.
  
## <a name="using-azure-infrastructure-services-for-internet-sites"></a>Azure-infrastructuur services gebruiken voor Internet sites

Microsoft Azure biedt een boeiende optie voor het hosten van Internet sites op basis van SharePoint Server 2013. Voordelen is onder meer het volgende:
  
- Richt u op het ontwikkelen van een fantastische site in plaats van de infrastructuur van gebouwen.
    
- Flexibiliteit voor het schalen van uw oplossing op basis van de vraag en het schalen.
    
- Betaal alleen voor de middelen die u nodig hebt.
    
- Profiteren van Azure Active Directory voor klant accounts.
    
- Functies toevoegen die momenteel niet beschikbaar zijn in Microsoft 365, zoals uitgebreide rapporten en analyses.
