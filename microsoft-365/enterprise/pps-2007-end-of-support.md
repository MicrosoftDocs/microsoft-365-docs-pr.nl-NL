# <a name="performancepoint-server-2007-end-of-support-roadmap"></a>Roadmap voor einde van ondersteuning voor PerformancePoint Server 2007

*Dit artikel is van toepassing op Microsoft 365 Enterprise en Office 365 Enterprise.*

Servers en toepassingen van Office 2007 hebben hun einde van de ondersteuning bereikt, waaronder servers en toepassingen die u gebruikt als onderdeel van uw BI-oplossingen (Business Intelligence). De volgende tabel bevat BI-toepassingen waarop dit van toepassing is:

Zie [bronnen om u te helpen bij het upgraden van Office 2007-servers en-clients](upgrade-from-office-2007-servers-and-products.md)voor meer informatie.
  
## <a name="what-does-end-of-support-mean"></a>Wat betekent *einde van ondersteuning* ?

Net zoals de meeste Microsoft-producten, PerformancePoint Server 2007 SP3, Proclarity-software en SharePoint Server 2007 SP3, hebben we een ondersteunings levenscyclus, waaronder Microsoft nieuwe functies, foutoplossingen en beveiligingsupdates biedt. De levensduur van een product duurt doorgaans tien jaar vanaf de eerste versie van het product. Het einde van de levenscyclus bekend is als het einde van de ondersteuning van dit product. Aangezien ProClarity, PerformancePoint Server en SharePoint Server 2007 hun einde van de ondersteuning hebben bereikt, biedt Microsoft niet langer:
  
- Technische ondersteuning voor problemen die kunnen optreden.
    
- Oplossingen voor ontdekte problemen die gevolgen kunnen hebben voor de stabiliteit en bruikbaarheid van servers.
    
- Beveiligingscorrecties voor ontdekte beveiligingsproblemen die servers of toepassingen kwetsbaar kunnen maken voor beveiligingsrisico's.
    
- Tijdzone-updates.
    
De installatie van ProClarity, SharePoint Server 2007 SP3 en PerformancePoint Server 2007 SP3 blijft ook actief, hoewel de ondersteuning is beëindigd. We raden u echter ten zeerste aan dat u zo snel mogelijk migreert van deze toepassingen.

### <a name="what-if-i-dont-upgrade-now"></a>Wat moet ik doen als ik nu niet upgrade?

U kunt ervoor kiezen om niet onmiddellijk een upgrade uit te voeren. Uw bestaande servers en toepassingen worden uitgevoerd. U ontvangt geen verdere updates, waaronder beveiligingsupdates, omdat ondersteuning is beëindigd. En als er iets mis is met de servertoepassingen, kunt u geen hulp krijgen van de technische ondersteuning van Microsoft.

### <a name="use-power-bi-in-the-cloud-or-on-premises"></a>Power BI (in de Cloud of on-premises) gebruiken

Power BI is een suite met bedrijfsanalyse Programma's voor het analyseren van gegevens en het delen van inzichten. Met Power BI kunt u on-premises of online gegevensbronnen gebruiken om interactieve rapporten en dashboards te maken. Personen kunnen uw rapporten en dashboards weergeven en gebruiken op hun computer of mobiele apparaat.
  
Power BI maakt geen deel uit van Microsoft 365 of SharePoint Server. Het is een aparte aanbieding die Power BI Desktop, Power BI-gateways en de Power BI-service omvat. Power BI is ook geïntegreerd met SharePoint Online. U kunt aan de slag met Power BI gratis. Op basis van uw gegevens gebruik en zakelijke behoeften kunt u later upgraden naar Power BI Pro met Microsoft 365 E5. Zie [Wat is Power bi?](https://go.microsoft.com/fwlink/?linkid=841341) voor meer informatie.
  
### <a name="use-reporting-services-on-premises"></a>Reporting Services (on-premises) gebruiken

SQL Server Reporting Services biedt een krachtige rapportage oplossing. U kunt Reporting Services configureren in de native modus of in de SharePoint-modus. U kunt verschillende hulpmiddelen gebruiken voor het maken van rapporten, waaronder rapport Designer, Report Builder en Power View. Met de nieuwste versie van SQL Server kunt u ook SQL Server Mobile Report Uitgever gebruiken om rapporten te bezorgen die naar een willekeurige schermgrootte schalen. Dit zorgt ervoor dat gebruikers rapporten op hun mobiele apparaat kunnen gebruiken. Zie [SQL Server Reporting Services (SSRS): mobiele en gepagineerde rapporten maken, implementeren en beheren](https://go.microsoft.com/fwlink/?linkid=841342)voor meer informatie.
  
### <a name="use-performancepoint-services-on-premises"></a>PerformancePoint Services (on-premises) gebruiken

PerformancePoint Server 2007 is apart verkocht bij SharePoint Server 2007. Vanaf SharePoint Server 2010 is PerformancePoint Services een servicetoepassing van SharePoint Server. U hoeft dus geen aparte server-licenties of-hardware te kopen voor het gebruik van PerformancePoint Services.
  
Als u wilt overstappen van PerformancePoint Server 2007 naar PerformancePoint Services, gaat u naar een nieuwere versie van SharePoint Server en configureert u PerformancePoint Services. De versie van SharePoint Server die u verschuift om te bepalen of u uw bestaande dashboardinhoud van PerformancePoint Server 2007 naar PerformancePoint Services kunt importeren.
  
- Als u een upgrade uitvoert naar SharePoint Server 2010, kunt u de inhoud van uw PerformancePoint-dashboard importeren van PerformancePoint Server 2007 in PerformancePoint Services in SharePoint Server 2010. Voor meer informatie raadpleegt u de [wizard importeren: inhoud van PerformancePoint server 2007 naar SharePoint server 2010](https://go.microsoft.com/fwlink/?linkid=838873).
    
- Als u overstapt op SharePoint Server 2013 of SharePoint Server 2016, moet u waarschijnlijk nieuwe dashboardinhoud maken (gegevensbronnen, rapporten, scorecards en dashboardpagina's).
    
Raadpleeg de volgende bronnen om aan de slag te gaan met uw upgrade plan voor PerformancePoint Services:
  
- [Einde van ondersteunings schema van SharePoint Server 2007](sharepoint-2007-end-of-support.md)
    
- Wanneer u weet naar welke versie van SharePoint u gaat overstappen, raadpleegt u het bijbehorende artikel voor PerformancePoint Services:
    
  - [Plannen voor PerformancePoint Services (SharePoint Server 2010)](https://go.microsoft.com/fwlink/?linkid=841363)
    
  - [Overzicht van PerformancePoint Services in SharePoint Server 2013](https://go.microsoft.com/fwlink/?linkid=841551)
    
  - [Overzicht van PerformancePoint Services in SharePoint Server 2016](https://go.microsoft.com/fwlink/?linkid=874704)
    
Wanneer u een upgrade uitvoert naar PerformancePoint Services, krijgt u diverse nieuwe functies en verbeteringen. PerformancePoint Services biedt verbeterde scorecards. nieuwe visualisaties, zoals de uitgevouwen structuur en het rapport met KPI-details. meer grafiektypen; betere functies voor time intelligence-filtering. en de toegankelijkheids naleving is verbeterd. Zie [Nieuw voor PerformancePoint Services (SharePoint Server 2010)](https://go.microsoft.com/fwlink/?linkid=841343)voor meer informatie.
