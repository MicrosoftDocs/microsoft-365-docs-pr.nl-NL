# <a name="how-to-manage-microsoft-365-with-windows-powershell-for-delegated-access-permissions-partners"></a>Microsoft 365 beheren met Windows PowerShell voor partners met gedelegeerde toegangsmachtigingen

*Dit artikel is van toepassing op Microsoft 365 Enterprise en Office 365 Enterprise.*

DAP-partners (Gedelegeerde Access Permission) zijn Syndication- en Cloud Solution Providers (CSP)-partners. Veel zijn netwerk- of telecomproviders. Ze bundelen Microsoft 365-abonnementen in hun serviceaanbiedingen. Wanneer ze een Microsoft 365-abonnement verkopen, krijgen ze automatisch machtigingen voor beheer namens (AOBO) aan de tenancies van de klant, zodat ze deze tenancies kunnen beheren en rapporteren. Deze taken zijn moeilijk te doen in het Microsoft 365-beheercentrum. Het is veel gemakkelijker om PowerShell voor Microsoft 365 te gebruiken om beheertaken uit te voeren, zoals:
- Alle **tenant-ids** en hun domeinen van de klant op een lijst zetten 
- Alle gebruikers in een klantentenancy en hun toegewezen licenties identificeren
> [!NOTE]
> Sommige beheertaken kunnen alleen worden uitgevoerd in PowerShell.
