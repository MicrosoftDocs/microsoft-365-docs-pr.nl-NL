# <a name="office-365-management-activity-api"></a>Office 365 Management Activity API

Microsoft biedt Reporting Services die u kunt gebruiken om een samenvoeging van de Office 365-Tenant te verkrijgen met samenvoeginformatie. De [Office 365 Management Activity API](https://docs.microsoft.com/office/office-365-management-api/office-365-management-apis-overview#office-365-management-activity-api) maakt gebruik van een industrie-standaard aanliggend ontwerp en OAuth v2 voor verificatie. Dit maakt het eenvoudig om te experimenteren met het ophalen van gegevens en het opzeggen van de opname Programma's en toepassingen. De API biedt een gegevensfeed met informatie over gebruikers, beheerders, bewerkingen en beveiligingsactiviteiten in Office 365. De gegevens kunnen ter regulering worden bewaard of gecombineerd met logboekgegevens van een on-premises infrastructuur of andere bronnen. Dit helpt bij het maken van een toezicht oplossing voor bewerkingen, beveiliging en compliance in de onderneming.

De Office 365 Management Activity API biedt informatie over diverse acties en gebeurtenissen van gebruikers, beheerders, systemen en beleid uit Office 365 en Azure Active Directory-activiteitenlogboeken. De API biedt een consistent controleschema met meer dan tien velden die in alle services worden gemeenschappelijk. Met de API kunnen organisaties eenvoudige verbindingen tussen gebeurtenissen aanbrengen en kunnen nieuwe manieren om redenen voor de gegevens worden ingeschakeld.

Tientallen onafhankelijke software leveranciers (Isv's) die zijn gekoppeld aan Microsoft en die oplossingen hebben gemaakt op basis van de API. Sommige oplossingen zijn alleen gericht op Office 365-gegevens en andere brongegevens van meerdere cloud providers en on-premises systemen, zodat een uniforme weergave wordt gemaakt van activiteiten, beveiliging en compliance-gerelateerde activiteiten. 

Zie voor meer informatie de [naslag voor Office 365 Management Activity API](https://docs.microsoft.com/office/office-365-management-api/office-365-management-activity-api-reference).
