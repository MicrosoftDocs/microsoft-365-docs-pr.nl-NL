# <a name="migrate-your-organization-data-to-microsoft-365-enterprise"></a>Gegevens van uw organisatie migreren naar Microsoft 365 Enterprise

## <a name="ready-to-migrate-your-data-to-microsoft-365"></a>Bent u klaar voor het migreren van uw gegevens naar Microsoft 365?

Onze [basisinstructies](https://support.office.com/article/Set-up-Office-365-for-business-6a3a29a0-e616-4713-99d1-15eda62d04fa) voor het instellen van uw microsoft 365-Services en-toepassingen kunnen het kortst mogelijke voor iedereen gebruiken. Dit slaat de hybride scenario's en de geavanceerdere methoden over voor de migratie naar Microsoft 365. 
  
Als u hulp nodig hebt bij het instellen van Microsoft 365, is [FastTrack](https://fasttrack.microsoft.com/office) de eenvoudigste manier om microsoft 365 te implementeren, u kunt zich ook aanmelden en de [Installatiehandleiding voor Microsoft 365-Services](setup-guides-for-microsoft-365.md)gebruiken.
