# <a name="getting-started-with-powershell-for-microsoft-365"></a>Aan de slag met PowerShell voor Microsoft 365

*Dit artikel is van toepassing op Microsoft 365 Enterprise en Office 365 Enterprise.*

Met PowerShell voor Microsoft 365 kunt u Microsoft 365 beheren met opdrachten en scripts om uw dagelijkse werk te stroomlijnen. In dit artikel leest u waarom Microsoft PowerShell voor Microsoft 365-vaardigheden essentieel zijn voor het beheren van Microsoft 365, hoe u verbinding maakt met uw Microsoft 365-abonnement, rapporten maakt en aanvullende informatie en hulp krijgt via de Microsoft 365-community.
  
Selecteer het artikel op basis van uw behoeften:
  
- [Waarom u PowerShell voor Microsoft 365 moet gebruiken](why-you-need-to-use-microsoft-365-powershell.md)
    
    Begin hier met de nieuwe functies in PowerShell voor Microsoft 365 en leer zes redenen waarom u PowerShell voor Microsoft 365 moet gebruiken. 
    
- [Verbinding maken met Microsoft 365 met PowerShell](connect-to-microsoft-365-powershell.md)
    
    Begin hier om verbinding te maken met uw abonnement op Microsoft 365 met behulp van PowerShell voor Microsoft 365 en beheertaken vanaf de opdrachtregel uit te voeren.
    
- [Verbinding maken met alle Microsoft 365-Services in één Windows PowerShell-venster](connect-to-all-microsoft-365-services-in-a-single-windows-powershell-window.md)
    
    U kunt Microsoft 365 in aparte vensters voor Skype voor bedrijven online, SharePoint Online, Microsoft Exchange Online en Microsoft 365 accounts en licenties beheren. U kunt deze ook beheren via één venster. In dit onderwerp wordt uitgelegd hoe u dit doet.
    
- [Windows PowerShell gebruiken om rapporten te maken in Microsoft 365](use-windows-powershell-to-create-reports-in-microsoft-365.md)
    
    Begin hier met de installatie van de PowerShell voor Microsoft 365-modules en u meer wilt weten over het gebruik van automatiserings opdrachten om snel rapporten te maken. 
    
- [Cmdlet-verwijzingen voor Microsoft 365-Services](cmdlet-references-for-microsoft-365-services.md)
    
    Begin hier om de onderwerpen te vinden die de cmdlets voor de modules PowerShell voor Microsoft 365 beschrijven.
    
- [PowerShell voor Microsoft 365 community-bronnen](microsoft-365-powershell-community-resources.md)
    
    Begin hier om verbinding te maken met de locaties van de PowerShell-Community en Help-informatie en informatie over het gebruik van PowerShell voor Microsoft 365.
