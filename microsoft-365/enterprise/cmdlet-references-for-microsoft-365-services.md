# <a name="cmdlet-references-for-microsoft-365-services"></a>Cmdlet-verwijzingen voor Microsoft 365-Services 

*Dit artikel is van toepassing op Microsoft 365 Enterprise en Office 365 Enterprise.*

Dit artikel bevat informatie over de cmdlets voor de diverse Microsoft 365-Services en de verbindings instructies voor elke Microsoft 365-service die door PowerShell wordt ondersteund.
  
> [!NOTE]
> Zie [verbinding maken met alle Microsoft 365-Services in één Windows PowerShell-venster](connect-to-all-microsoft-365-services-in-a-single-windows-powershell-window.md)om verbinding te maken met alle services tegelijk.
  
## <a name="azure-active-directory-powershell-cmdlets"></a>Azure Active Directory PowerShell-cmdlets

De naslaginformatie voor de Azure Active Directory PowerShell voor Graph-cmdlets vindt u in het gedeelte verwijzingen van de [documentatie van Azure Active Directory PowerShell voor Graph](https://docs.microsoft.com/powershell/azure/active-directory/install-adv2?view=azureadps-2.0).

De naslaginformatie voor de Azure Active Directory-module voor Windows PowerShell-cmdlets vindt u in het gedeelte verwijzingen van de [Azure Active Directory-documentatie (MSOnline)](https://docs.microsoft.com/powershell/azure/active-directory/overview?view=azureadps-1.0).

Zie [verbinding maken met Microsoft 365 via PowerShell](connect-to-microsoft-365-powershell.md)voor microsoft 365 PowerShell-verbindings instructies.
  
## <a name="exchange-online-powershell-cmdlets"></a>PowerShell-cmdlets voor Exchange Online

Naslaginformatie over de cmdlets voor Exchange Online voor cmdlets vindt u in het gedeelte verwijzingen van de [Exchange Online PowerShell-documentatie](https://docs.microsoft.com/powershell/exchange/exchange-online-powershell).
  
Zie [verbinding maken met Exchange Online PowerShell](https://go.microsoft.com/fwlink/p/?LinkId=396554)voor verbindings instructies voor Exchange Online PowerShell.
  
> [!NOTE]
> Cmdlets voor rapporten voor andere services, zoals SharePoint Online, Skype voor bedrijven online en Microsoft 365 User Activity, zijn beschikbaar in Exchange Online PowerShell. Zie cmdlets voor het [rapporteren van Exchange Online](https://go.microsoft.com/fwlink/p/?LinkId=691595)voor meer informatie. 
  
## <a name="sharepoint-online-powershell-cmdlets"></a>PowerShell-cmdlets voor SharePoint Online

Zie [index van Windows PowerShell voor SharePoint Online-](https://go.microsoft.com/fwlink/p/?LinkId=691476)cmdlets voor cmdlets voor SharePoint Online-cmdlets.
  
Zie [de Windows PowerShell-omgeving voor SharePoint Online Management Shell instellen voor de](https://go.microsoft.com/fwlink/p/?LinkId=691603)verbindings instructies voor SharePoint Online PowerShell.
  
## <a name="skype-for-business-online-powershell-cmdlets"></a>PowerShell-cmdlets voor Skype voor bedrijven online

Zie [cmdlets voor Skype voor bedrijven online](https://technet.microsoft.com/library/mt228132.aspx)voor informatie over de cmdlets voor Skype voor bedrijven online.
  
Zie [Skype voor bedrijven online beheren met PowerShell](manage-skype-for-business-online-with-microsoft-365-powershell.md)voor de verbindings instructies voor Skype voor bedrijven online PowerShell.

## <a name="security--compliance-center-powershell-cmdlets"></a>PowerShell-cmdlets voor beveiliging & nalevings centrum

De informatie over de cmdlets voor het beveiligings & voor compliance Center bevindt zich in de sectie verwijzingen van [& de PowerShell-documentatie over naleving](https://docs.microsoft.com/powershell/exchange/scc-powershell)
  
Zie [verbinding maken met de PowerShell voor naleving van beveiligings & voor de](https://docs.microsoft.com/powershell/exchange/connect-to-scc-powershell)verbindings instructies voor de beveiligings & nalevings centrum PowerShell.
