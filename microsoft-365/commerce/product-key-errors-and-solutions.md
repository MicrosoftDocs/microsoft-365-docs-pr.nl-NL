# <a name="problems-with-your-microsoft-365-for-business-product-key"></a>Hebt u problemen met uw productcode voor Microsoft 365 voor bedrijven?

Als er een foutbericht wordt weergegeven wanneer u een productcode voor **Microsoft 365 for Business**probeert in te wisselen, kunt u deze veelgebruikte oplossingen proberen. 
  
 **Hebt u onmiddellijk hulp nodig?** [Neem contact op Microsoft ondersteuning](../admin/contact-support-for-business-products.md). 
  
 Zie **voor hulp bij een Microsoft 365-serie of Microsoft 365 Personal-productcode voor meer informatie over het** [gebruik van productcodes met Office](https://support.microsoft.com/office/12a5763a-d45c-4685-8c95-a44500213759.aspx).
