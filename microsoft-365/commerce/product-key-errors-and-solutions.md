# <a name="problems-with-your-microsoft-365-for-business-product-key"></a>Problemen met uw Microsoft 365 productcode voor bedrijven?

Als er een foutbericht wordt weergegeven wanneer u probeert een productcode in te wisselen voor Microsoft 365 **voor** bedrijven, probeert u deze algemene oplossingen.
  
 **Hebt u onmiddellijk hulp nodig?** [Bel Microsoft-ondersteuning.](../business-video/get-help-support.md)
  
 **Zie Productcodes** gebruiken met Microsoft 365 Family of Microsoft 365 Personal productcode voor hulp bij [Office.](https://support.microsoft.com/office/12a5763a-d45c-4685-8c95-a44500213759.aspx)

|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
