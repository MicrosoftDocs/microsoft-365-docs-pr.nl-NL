# <a name="microsoft-managed-desktop-supported-regions-and-languages"></a>Ondersteunde regio's en talen van Microsoft Managed Desktop

In dit onderwerp vindt u informatie over regio's en talen die worden ondersteund door Microsoft Managed Desktop. 

- **Regio's:** Verenigde Staten, Canada (met uitzondering van Quebec), Verenigd Koninkrijk, België, Luxemburg, Nederland, Australië en Nieuw-Zeeland
- **Talen:** Engels is de enige ondersteunde taal die momenteel beschikbaar is voor gebruikers op beheerde apparaten. Dit is van toepassing op zowel [de ondersteuningsopties](https://docs.microsoft.com/microsoft-365/managed-desktop/working-with-managed-desktop/admin-support) voor beheerders als de ondersteuningsopties voor [eindgebruikers.](https://docs.microsoft.com/microsoft-365/managed-desktop/working-with-managed-desktop/end-user-support) 

Klanten kunnen echter nog steeds beheerde apparaten buiten de hierboven vermelde regio's gebruiken zonder een onderbreking van de Microsoft Managed Desktop-service. Een werknemer uit het Verenigd Koninkrijk kan bijvoorbeeld veilig werken en updates ontvangen over zijn beheerde apparaat tijdens een reis door Azië, Europa of Zuid-Amerika.

Zie ondersteuning voor Microsoft Managed Desktop [voor](https://docs.microsoft.com/microsoft-365/managed-desktop/service-description/support)meer informatie over gebruikersondersteuning met Microsoft Managed Desktop.

Als een gebruiker liever in een andere taal dan het Engels werkt, kan hij of zij zijn lokale IT-beheerder vragen om extra taalpakketten op beheerde apparaten centraal te implementeren via de Intune Company Portal-app. Vanaf Windows 10, versie 1803, worden taalpakketten geleverd als appx-pakketten in de Microsoft Store. Zie [Microsoft Store voor Bedrijven-apps](https://docs.microsoft.com/microsoft-365/managed-desktop/get-started/deploy-apps#msfb-apps)voor meer informatie.


>[!NOTE]
>De GetHelp-app op een Microsoft Managed Desktop-apparaat ondersteunt alleen Engels. Als gebruikers hulp kunnen krijgen bij Microsoft Managed Desktop via de GetHelp-app, moeten ze de weergavetaal overschakelen naar Engels.
