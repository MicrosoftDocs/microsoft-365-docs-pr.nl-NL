# <a name="microsoft-managed-desktop-devices"></a>Microsoft Managed Desktop-apparaten 

Microsoft Managed Desktop evalueert regelmatig apparaten die in de service moeten worden opgenomen. In dit onderwerp worden de vereisten beschreven voor een apparaat dat moet worden ingeschreven in Microsoft Managed Desktop en worden de specifieke apparaten weergegeven die momenteel worden ondersteund.

Zie [Microsoft Managed Desktop-documentatie en -bronnen voor](https://docs.microsoft.com/microsoft-365/managed-desktop/)meer informatie over Microsoft Managed Desktop.

Als u wilt deelnemen aan Microsoft Managed Desktop, moet een apparaat een van de volgende modellen zijn en voldoen aan of hoger zijn dan de vermelde specificaties voor RAM, processorfamilie en schijfruimte. 

De links naar apparaten hier zijn alleen voor uw referentie. Als u apparaten wilt bestellen, werkt u samen met uw commerciële kanaalcontacten om ervoor te zorgen dat u de juiste configuraties kiest. In deze tabel is de *archiefdatum* de datum waarop deze modellen worden onderworpen aan een kortere microsoft Managed Desktop-serviceduur; dergelijke apparaten die na deze datum zijn ingebruikgenomen, ontvangen geen volledige ondersteuningstermijn van drie jaar van Microsoft Managed Desktop. Gearchiveerde apparaten worden ook vermeld in [gearchiveerde apparaten](archived-device-list.md)van Microsoft Managed Desktop. *De pensioendatum* is de datum waarop dit model helemaal niet meer wordt ondersteund door Microsoft Managed Desktop. Op die datum worden alle ingeschreven apparaten van dit model verwijderd van Microsoft Managed Desktop, ongeacht wanneer ze zijn ingeschreven.
