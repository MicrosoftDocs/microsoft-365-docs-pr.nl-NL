# <a name="microsoft-managed-desktop-archived-devices"></a>Gearchiveerde apparaten met Microsoft Managed Desktop

Om de beste gebruikerservaring te garanderen, wordt op een bepaald moment de ondersteuning voor apparaten door Microsoft Managed Desktop ingetrokken. In dit artikel worden apparaten beschreven die de komende paar jaar bijna niet meer worden gebruikt, wat we 'gearchiveerd' noemen. Als u aan de slag gaat met Microsoft Managed Desktop met apparaten die u al hebt, moet u deze lijst zorgvuldig controleren.

>[!IMPORTANT]
>U kunt deze apparaten vandaag nog registreren, maar ze hebben een kortere levensduur met Microsoft Managed Desktop. Als u nieuwe apparaten koopt, moet u modellen gebruiken uit de [lijst met actieve apparaten.](./device-list.md)

Als u wilt worden geregistreerd in Microsoft Managed Desktop, moet een apparaat een van de volgende modellen zijn en voldoen aan de vermelde specificaties voor RAM, processorfamilie en schijfruimte. Er worden geen andere aanpassingen ondersteund.



>[!NOTE]
>In deze tabel is de 'retirement date' de datum waarop dit model helemaal niet meer wordt ondersteund door Microsoft Managed Desktop. Op die datum worden alle geregistreerde apparaten van dit model verwijderd uit het beheerde bureaublad van Microsoft, ongeacht wanneer ze zijn geregistreerd.
