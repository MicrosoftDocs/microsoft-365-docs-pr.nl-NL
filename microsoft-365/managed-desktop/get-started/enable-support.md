# <a name="enable-user-support-features"></a>Ondersteuningsfuncties voor gebruikers inschakelen

Of u nu uw eigen gebruikersondersteuning biedt of met een partner werkt om deze te leveren, volg deze stappen om de ondersteuningsprovider in staat te stellen om verhoogde apparaattoegang aan te vragen of problemen te escaleren naar Microsoft Managed Desktop, indien nodig.

1. Als ze er nog geen hebben, hebben gebruikers een account nodig in hetzelfde domein Azure Active Directory (AAD) als de Microsoft Managed Desktop apparaten.
2. Voeg de gebruikersaccounts van stap 1 toe aan de **beveiligingsgroep Moderne Roles-Support partner** in AAD.
