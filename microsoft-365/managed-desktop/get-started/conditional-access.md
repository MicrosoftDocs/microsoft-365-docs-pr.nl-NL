# <a name="adjust-conditional-access"></a>Voorwaardelijke toegang aanpassen

Als u een beleid voor [voorwaardelijke toegang](https://docs.microsoft.com/azure/active-directory/conditional-access/overview) in uw organisatie gebruikt, moet u instellen dat ze bepaalde accounts uitsluiten, zodat Microsoft Managed Desktop goed kan werken.

Volg deze stappen om dit te doen:

1. Raadpleeg het gedeelte 'Terugdraaien- stappen' van [Hoe: Uw voorwaardelijke toegang-implementatie plannen in Azure Active Directory](https://docs.microsoft.com/azure/active-directory/conditional-access/plan-conditional-access#rollback-steps).
2. Volg de stappen daar om de groep *Moderne Workplace Service Accounts* voor alle beleidsregels uit te sluiten.


Als u problemen hebt met voorwaardelijke toegang, neemt u contact op met de [beheerdersondersteuning](../working-with-managed-desktop/admin-support.md).
