# <a name="set-up-microsoft-managed-desktop-devices"></a>Microsoft Managed Desktop-apparaten instellen

U kunt zowel nieuwe als bestaande apparaten gebruiken in het Microsoft-beheer bureaublad.

## <a name="to-obtain-new-devices"></a>Nieuwe apparaten verkrijgen

We raden u aan een van onze goedgekeurde apparatuur te gebruiken. U kunt met uw Microsoft-account contactpersoon werken voor meer hulp bij het instellen van een apparaat-partnership. In essentie is dit echter wel zo:

1. Bekijk de lijst met [apparaten die momenteel zijn goedgekeurd](../service-description/device-list.md).
2. Volg de apparaten die werken met een goedgekeurde apparaat-partner.
3. Ga op een van de volgende manieren te werk om een van de volgende handelingen uit te voeren:
    - [Nieuwe apparaten zelf registreren](register-devices-self.md)
    - Werken met een partner om de apparaten te registreren
4. [Uw gebruikers voorbereiden op de](get-started-devices.md) gebruik van Microsoft beheerde bureaublad apparaten

## <a name="to-reuse-existing-devices"></a>Bestaande apparaten opnieuw gebruiken

1. Bekijk de lijst met [momenteel goedgekeurde apparaten](../service-description/device-list.md) om te bepalen welke van uw apparaten is goedgekeurd voor gebruik met Microsoft Managed Desktop. Als u een bestaand apparaat opnieuw gebruikt, moet u het opnieuw maken.
2. Voer een van de volgende handelingen uit:
    - [Bestaande apparaten zelf registreren](register-reused-devices-self.md)
    - Werken met een partner om de apparaten te registreren
3. [Uw gebruikers voorbereiden op de](get-started-devices.md) gebruik van Microsoft beheerde bureaublad apparaten
