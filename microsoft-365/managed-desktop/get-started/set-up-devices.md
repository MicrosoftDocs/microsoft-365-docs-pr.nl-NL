# <a name="set-up-microsoft-managed-desktop-devices"></a>Beheerde bureaubladapparaten van Microsoft instellen

U zowel nieuwe als bestaande apparaten gebruiken in Microsoft Managed Desktop.

## <a name="to-obtain-new-devices"></a>Nieuwe apparaten verkrijgen

We raden u aan om samen te werken met een van onze partners voor goedgekeurde apparaten. U met uw Microsoft-accountcontact werken voor meer hulp bij het opzetten van een apparaatpartnerschap. In wezen is het proces echter als volgt:

1. Bekijk de lijst met [momenteel goedgekeurde apparaten](../service-description/device-list.md).
2. Bestel de apparaten en werk samen met een goedgekeurde apparaatpartner.
3. Zodra ze zijn aangekomen, doe een van de volgende:
    - [Registreer zelf nieuwe apparaten](register-devices-self.md)
    - Werk samen met een partner om de apparaten te registreren
4. [Uw gebruikers klaar maken](get-started-devices.md) voor gebruik van Microsoft Managed Desktop-apparaten

## <a name="to-re-use-existing-devices"></a>Bestaande apparaten opnieuw gebruiken

1. Bekijk de lijst met [momenteel goedgekeurde apparaten](../service-description/device-list.md) om te bepalen welke van uw apparaten is goedgekeurd voor gebruik met Microsoft Managed Desktop. Als u een bestaand apparaat opnieuw gebruikt, moet u het opnieuw bekijken.
2. Ga uit van een van de volgende handelingen:
    - [Bestaande apparaten zelf registreren](register-reused-devices-self.md)
    - Werk samen met een partner om de apparaten te registreren
3. [Uw gebruikers klaar maken](get-started-devices.md) voor gebruik van Microsoft Managed Desktop-apparaten
