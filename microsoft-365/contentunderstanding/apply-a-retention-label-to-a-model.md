# <a name="apply-a-retention-label-to-a-document-understanding-model-preview"></a>Een Bewaar label toepassen op een document wat is het model (voorbeeld)

> [!Note] 
> De inhoud in dit artikel is bedoeld voor project cortex private preview. [Lees meer over project cortex](https://aka.ms/projectcortex).

</br>

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE4CSoL]

</br>

U kunt eenvoudig een [Bewaar label](https://docs.microsoft.com/microsoft-365/compliance/retention?view=o365-worldwide#retention-labels) toepassen op een document met beinformatie over model in Project cortex.

Met labels voor bewaarbeleid kunt u bewaarinstellingen toepassen op de documenten die het document identificeren.  U kunt bijvoorbeeld het model alleen voor het verlengen van documenten identificeren dat naar uw documentbibliotheek wordt geÃ¼pload, maar u kunt ook een Bewaar label toepassen, zodat deze documenten nooit kunnen worden verwijderd.

U kunt een Bewaar label toepassen op uw document met behulp van uw modelinstellingen. 

> [!Note]
> Voor Bewaar etiketten die beschikbaar zijn voor uw model, moeten ze worden [gemaakt en gepubliceerd in het nalevings centrum van Microsoft 365](https://docs.microsoft.com/microsoft-365/compliance/create-apply-retention-labels?view=o365-worldwide#how-to-create-and-publish-retention-labels).
