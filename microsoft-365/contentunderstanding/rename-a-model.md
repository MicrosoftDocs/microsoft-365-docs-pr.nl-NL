# <a name="rename-a-model-in-microsoft-sharepoint-syntex"></a>De naam van een model wijzigen in Microsoft SharePoint Syntex

Op een bepaald moment wilt je misschien de naam van een model met documentbegrip wijzigen. Een veelvoorkomende voorbeeld hiervan is wanneer je bij het eerste concept van een model misschien niet veel aandacht hebt besteed aan de definitieve naam (je hebt het model bijvoorbeeld 'AlexWilburModel1' genoemd). Wanneer je het model bijna hebt voltooid en in gebruik wilt nemen, besef je dat 'Contractvernieuwingen' een geschiktere naam zou zijn en dat je de naam van het model wilt wijzigen.  

Een ander voorbeeld is wanneer je organisatie besluit om met een andere naam te verwijzen naar een proces of documenttype. Nadat je je model hebt gemaakt en klaar bent om het toe te passen, kan je organisatie bijvoorbeeld bepalen dat alle 'Contracten' nu officieel 'Overeenkomsten' worden genoemd. Indien nodig kunt je de naam van je model wijzigen van 'Contractverlenging' in 'Overeenkomstverlenging'.

> [!IMPORTANT]
> Je kunt de naam van een model met documentbegrip alleen wijzigen als het model niet op een documentbibliotheek is toegepast. 

Als je de naam van een model wijzigt, wijzig je ook de naam van het [inhoudstype](/sharepoint/governance/content-type-and-workflow-planning#content-type-overview) dat aan het model is gekoppeld.

## <a name="rename-a-model"></a>De naam van een model wijzigen

Volg deze stappen om de naam van een model met documentbegrip te wijzigen.

1. Selecteer in het inhoudscentrum **Modellen** om de lijst met modellen te zien.

2. Selecteer op de pagina **Modellen** het model waarvan je de naam wilt wijzigen.

3. Gebruik het lint of de knop **Acties weergeven** (naast de modelnaam) om **Naam wijzigen** te selecteren. </br>

    ![Schermafbeelding van de pagina Modellen met een geselecteerd model waarbij de opties voor het wijzigen van de modelnaam zijn gemarkeerd.](../media/content-understanding/select-model-rename-both.png) </br>

4. In het deelvenster **Naam van model wijzigen**:

   a. Voer bij **Nieuwe naam** de nieuwe naam in van het model waarvan je de naam wilt wijzigen.</br>

    ![Schermafbeelding van het deelvenster Naam wijzigen.](../media/content-understanding/rename-model-panel.png) </br>

   b. (Optioneel) Selecteer bij **Geavanceerde instellingen** of je een bestaand [inhoudstype](/sharepoint/governance/content-type-and-workflow-planning#content-type-overview) wilt koppelen. Als je **Een bestaand inhoudstype gebruiken** kiest, wordt de naam van het model gewijzigd en afgestemd op het geselecteerde inhoudstype.

5. Selecteer **Naam wijzigen**.
